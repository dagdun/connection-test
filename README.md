## install
`yarn install`
or
`npm i`

## how to use

### mongodb
test connect default on localhost
`node index.js mongodb`

test to another machine
`node index.js mongodb://123.123.123.123:27017`

### rabbitmq
test connect default on localhost
`node index.js amqp`

test to another machine
`node index.js amqp://123.123.123.123:5672`
